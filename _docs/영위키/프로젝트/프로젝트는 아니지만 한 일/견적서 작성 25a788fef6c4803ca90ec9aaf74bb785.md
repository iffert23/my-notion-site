데이터 전송 요금은 어떻게 계산되나요?
Azure에 진입하는 트래픽에는 요금이 청구되지 않습니다. Azure에서 나가는 트래픽(VPN, ExpressRoute 또는 지점 및 사이트 간 사용자 VPN 연결을 통해)에는 표준 Azure 데이터 전송 요금 또는 ExpressRoute의 경우 ExpressRoute 가격 책정이 적용됩니다. .

vWAN 허브에 연결된 VNet이 데이터를 보내거나 받을 때 피어링 요금이 적용됩니다. 자세한 내용은 가상 네트워크 가격 책정을 참조하세요.

Virtual WAN 허브와 원본 허브가 아닌 다른 지역에 있는 원격 Virtual WAN 허브 또는 VNet 간의 데이터 전송 요금에 대해서는 허브에서 나가는 트래픽에 대한 데이터 전송 요금이 적용됩니다. 예: 미국 동부 허브에서 나가는 트래픽은 미국 서부 허브로 이동하는 데 $0.02/GB의 요금이 청구됩니다. 미국 서부 허브로 들어가는 트래픽에 대해서는 요금이 청구되지 않습니다.

모든 허브 간 트래픽은 지역 간(지역 내/대륙 간) 요금인 Azure 데이터 전송 요금 적용 대상입니다.
