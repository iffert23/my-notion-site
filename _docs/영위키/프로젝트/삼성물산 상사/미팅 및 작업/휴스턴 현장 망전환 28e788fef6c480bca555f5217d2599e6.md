# 휴스턴 현장 망전환

상태: 완료
날짜: 2025년 10월 16일   (GMT+9)

휴스턴 현장 VPN 사이트 추가, 설정

현장이 광고하는 BGP 대역 확인 (광고받는, 광고중인 대역 확인하는거임)

→ vWAN → vHub → VPN → 사이트선택 → BGP 대시보드 → 상단에 광고하는 경로/학습중인 경로 → 학습 경로 다운로드 → 엑셀에 맨 첫번째 행 필터걸고 필요한 IP나 포트 필터링 → 출발지-도착지 순으로 있는 포트 넘버 확인

Iperf3, RTT 용 NSG 추가

각  VPN GW에만 모니터링 - 진단 설정 추가
