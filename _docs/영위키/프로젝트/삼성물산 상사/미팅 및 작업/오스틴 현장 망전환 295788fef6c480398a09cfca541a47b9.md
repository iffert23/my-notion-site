# 오스틴 현장 망전환

상태: 완료
날짜: 2025년 10월 22일 → 2025년 10월 23일

오스틴 현장 망전환 관련해서는 크게 작업할 부분이 없었음

기존에 미리 오스틴 사이트 만들어두고 링크 설정까지 다 끝내놨기 때문임

근데 오스틴 백업 테스트 하다가 발견한 이슈

1. 원래 메인관문 뉴저지임
2. 근데 오스틴이 휴스턴을 메인관문처럼 사용함
3. 라우팅 테이블(미중남부, 미동부허브 유효경로) 확인해보니까 미중남부 허브쪽이랑 미동부 허브쪽에서 미주녹스 두개대역이 각각의 허브 광고로 두 대역이 두번씩 떠야되는데 한쪽 허브의 광고대역만 뜸(미중남부에서 보면 미중남부가 광고하는 미주녹스 대역과 미동부에서 광고하는 미주녹스 대역이 둘 다 떠야 되는데 미중남부(6100 6100)가 광고하는 미주녹스 대역만 뜨고 미동부(6100)에서 광고하는 미주녹스 대역이 안뜸)
4. 근데 메인 관문이 뉴저지이기 때문에 오스틴이 휴스턴이 아닌 뉴저지를 바라봐야 함
5. 이걸 해결하려면 BGP 광고 경로를 경로 맵을 사용해서 프리픽스를 /25에서 /26으로 쪼개거나 뉴저지 방화벽에서 수동으로 프리픽스를 쪼개서 BGP 광고때려야함
6. 그래서 경로 맵에서 프리픽스 쪼개는 테스트를 해봄 → 잘 안됨 → SR에 티겟올림(문의내용 : 원래 허브 광고가 겹치는 대역은 광고받지 않는게 맞는지, 경로 맵 프리픽스 쪼개는 기능 사용할 수 있는지)
