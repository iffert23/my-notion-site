- 설치하기 & 네트워크 연결하기& ifconfig, nslookup 까지
    1. 구글에 VMware WorkStation을 설치한다.
    2. Linux minimal ios를 검색해서 미니멀 ios를 다운받는다.  (확장자가 ios임 이상한 거 받지 마)
    3. VMware WorkStation을 설치한다 ios는 minimal로 선택.
    4. 설치되면 껍데기만 설치된 느낌. ifconfig가 안된다. (충격)
    5. 네트워크가 될 수 있게 ip부터 nmtui로 ip를 automatic으로 설정해놓는다.
    6. 인터페이스명은 ip addr이나 nmtui에서 확인할 수 있다.
    7. vi /etc/sysconfig/network-scripts/ifcfg-(인터페이스명) 을 한다.
    8. ONBOOT=no 부분을 ONBOOT=yes 로 설정하고 저장하고 나온다.
    9. systemctl restart network 를 한다.
    10. ifconfig와 nslookup 등등의 command를 설치하기 위해 yum install net-tools bind-utils -y 를 입력한다. (낑겨서 bind 까지 받아도 된다. → dns임)
    11. 그럼 인터넷은 된다! ping도 간다.
- ssh 연결하기
    1. yum install -y openssh-server openssh-clients openssh-askpass 로 ssh 패키지를 설치한다.
    2. vi /etc/ssh/sshd_config 에서 #port=22 에서 주석처리를 빼준다.
    3. firewall-cmd —permanent —add-port=22/tcp
    4. firewall-cmd —permanent —add-service=ssh
    5. 방화벽 reload 해주기
    6. PuTTY 등의 프로그램을 통해 ssh 가능한 것 확인하기.
