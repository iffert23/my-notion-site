# 클라우드 개념

프라이빗 클라우드 : 하나의 회사만을 위한 클라우드 환경. 회사 내부의 데이터센터 등

퍼블릭 클라우드 : 클라우드 서비스를 구매하려는 모든 사용자가 리소스에 엑세스하고 사용할 수 있음. Azure나 AWS 등의 외부 클라우드 서비스

하이브리드 클라우드 : 프라이빗 클라우드와 퍼블릭 클라우드를 결합한 형태. 민감하고 보안이 중요한 정보는 프라이 클라우드에, 외부 트래픽 제어같은 경우에는 퍼블릭 클라우드에 저장한다.

다중 클라우드 : 여러 퍼블릭 클라우드를 사용하는 환경

IaaS(Infrastructure as a Service), PaaS(Platform as a Service) 및 SaaS(Software as a Service) 개념

IaaS : 소비자에게 가장 큰 책임을 지우며, 클라우드 공급자는 물리적 보안, 전원 및 연결의 기본 사항을 담당

PaaS : 클라우드 공급자와 소비자 간에 책임을 균등하게 분산합니다. (IaaS와 SaaS의 중간 지점인)

SaaS : 클라우드 공급자가 대부분의 책임을 맡습니다. (IaaS의 반대)
