2. AOAI 리소스 생성
    - Azure OpenAI 리소스 활용 (Korea Central 안됨)
    - 인덱스, blob 등등 추가
3. 테스트 환경 구축
    - Azure Open AI Studio / Postman / Curl 사용해서 API 호출 테스트 하기
4. 테스트 시나리오 (테스트 목적을 충족하는지)
    - 문서 요약 하는지
    - 여러 방법으로 뭐 물어보면 대답 잘 하는지
