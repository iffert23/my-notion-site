- 삼성물산 로그확인
    
    IKE 확인 로그
    AzureDiagnostics
    | where Category == "IKEDiagnosticLog"
    //| where instance_s == "GatewayTenantWorker_IN_0"
    | where Message contains "Remote (해당 사이트IP):500"
    | project TimeGenerated, Category, ResourceType, OperationName, Level, Message, instance_s
    | order by TimeGenerated desc
    
    터널 확인 로그 (주석 제거하면 DPD도 확인가능)
    
    AzureDiagnostics
    | where Category == "TunnelDiagnosticLog"
    | where remoteIP_s contains "(해당 사이트IP)"
    //| where stateChangeReason_s contains "DPD timed out"
    | project TimeGenerated, Category, Level, OperationName, status_s, stateChangeReason_s, remoteIP_s, instance_s
    | order by TimeGenerated desc
