경로지정/경계경로 프로토콜

인터넷상에서 데이터가 빠르고 효율적으로 전달될 수 있도록 하는 라우팅 프로토콜

**hop counting 방식**
hop 수가 적을수록 빠르고 효율적인 경로로 인식함

데이터 라우팅을 통해 인터넷을 작동시킴

인터넷 : 소규모 네트워크의 집합체

**AS : 단일 관리 기관에서 제어하는 소규모 네트워크 (자율 시스템)**

*최적 경로 탐색*

*네트워크 연결 변경 사항 검색*

*네트워크 정책관리*

*네트워크 보안 계층 추가(rpki에 관한 내용인듯)*

**작동원리 : Peering**

- 특정 라우터를 Peer/스피커 라우터로 할당
- AS의 경계 또는 엣지에 있는 라우터(디바이스)
    - 경로 검색 : 인접한 피어와 라우팅 정보를 교환함
    - 경로 저장 : 경로 알림정보 수집 후 라우팅테이블 형태로 저장(keep alive)
    - 경로 선택 : 저장된 경로를 기반으로 최적,최우선 트래픽 라우팅(최단경로)

**유형**

iBGP : 내부BGP

eBGP : 외부BGP

**인터넷 확장관리 및 수용방법**

*Route Reflector(경로 리플렉터)*

*Confederation(컨페더레이션)*

*Route aggregation(경로 어그리게이션)*
