# SNMP 설정

---

1. snmp 활성화(enable)
snmp-server vrf <NAME>
1-1) snmp-server vrf mgmt	<-mgmt interface
1-2) snmp-server vrf default	<-default interface
1-3) snmp-server vrf <USERDEFINED_VRF_NAME>	<-user created vrf
2. 스위치에 대한 정보 설정 (임의설정이긴 한데 관리자 계정그룹 설정같음)
snmp-server system-contact <contact1>
snmp-server system-location <location2>
snmp-server system-description <SWname>

2-1 필요한 경우 snmp 기본 포트 변경
snmp-server agent-port

1. 커뮤니티 문자열 생성
snmp-server community
2. 트랩 수신기 구성
snmp-server host <host IP>
ex) snmp-server host <10.10.10.10> trap version <v2c> <community> <public>
3. 확인 명령어
show snmp agent-port
show snmp community
show snmp system
show snmpv3 context
show snmp trap	<-displays snmp alarm configuration
show snmp vrf	<-Who SNMP agent service is running
show snmpv3 users
show tech snmp

---

<예시>
`snmp-server vrf default
snmp-server system-description nac snmp
snmp-server system-location nac
snmp-server system-contact boboo
snmp-server community public`
