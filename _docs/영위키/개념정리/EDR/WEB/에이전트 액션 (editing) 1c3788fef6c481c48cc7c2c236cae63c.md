# 에이전트 액션 (editing)

---

- **Hostname 변경**
    1. **[정책] - [노드 액션]** 생성 (테스트 시 조건은 인증 사용자로 생성. 필요 시 해당되는 조건 설정)
    2. 호스트 이름 설정 (매크로 활용)
    3. 해당되는 사용자가 있는 노드 정책에 노드 액션 할당
    4. 노드 선택 후 노드 정책 즉시 적용
    5. 액션 수행 결과
        
        ![Untitled](../../NAC/WEB/%EC%97%90%EC%9D%B4%EC%A0%84%ED%8A%B8%20%EC%95%A1%EC%85%98%20(editing)/Untitled.png)
        
- **프로세스 강제 종료**
    1. **[노드 액션]** 생성
    2. **조건연산**(OR/AND), **조건설정**(프로세스, 동작하면, 프로세스명.exe), **플러그인 선택**(프로세스 강제 종료), **플러그인 설정**(팝업여부, 메시지 설정), **수행타입** 설정
    3. **[노드정책]** 생성 또는 기존 정책 수정
    4. **노드그룹 할당**, 세부설정,에이전트정책,관리정책(넘어가도 됨), **노드 액션 할당**
    5. **[관리] - [노드] - 모든 노드 선택 - 노드 대상 명령 - 노드 대상 작업지시 - 에이전트액션 즉시수행**
    6. 액션 수행 결과
        
        ![Untitled](../../NAC/WEB/%EC%97%90%EC%9D%B4%EC%A0%84%ED%8A%B8%20%EC%95%A1%EC%85%98%20(editing)/Untitled%201.png)
        
- **제어정책 (수행조건 검사 에이전트액션)**
    1. 해당 액션을 적용할 **[노드 그룹(정책전용x)]** 선택 또는 생성
    2. **그룹 조건** - **조건연산**(OR/AND), **조건설정**(에이전트 액션, 특정 액션을 만족하면/만족하지 않으면(수행전 포함), 액션선택), 추가
    3. **[제어정책]** 생성 - **[노드그룹] 할당**
    4. **[노드 정책]** - **[노드 액션] 할당**
    5. 정책 생성 결과 (메모장 미 동작 중이면 차단)
