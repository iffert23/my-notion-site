# 스위치 SNMP 수집

---

1. 센서에서 스위치로 snmpwalk 요청하고 리턴 확인
    
    `@shell`
    
    `snmpwalk -v2c -c public [SW IP] .1.3.6.1.2.1.17.1.1`
    
    *리턴값이 없다면(cannot -) BRIDGE-MIB이 지원되지 않는 스위치이기 때문에  SNMP 수집이 불가능하다.
    
2. 스위치에 snmp 설정되어있는지 확인
3. community값 확인 
4. **[설정] - [환경설정] - [노드관리]**
5. snmp 인증 설정에서 community 값 설정
6. 네트워크 정보 수집 on
7. 즉시수행
8. **[관리] - [노드] - [모든 노드 선택 또는 스위치 IP 선택] - [작업선택] - [노드 대상 명령] - [노드 대상 작업지시] - [노드 상세정보 스캐닝 수행]**
9. 해당 노드 클릭해서 **[네트워크정보] - [탐지된 서비스 목록]** 에 snmp 서버 있는지 확인
10. **[관리] - [스위치]** 에 스위치 정보 수집 되는지 확인

[노드 정보검사 설정하기 — Genians Documentation 5.0.57
 documentation](https://docs.genians.com/release/ko/system/check-node-information.html)

[스위치 찾아보기 — Genians Documentation 5.0.57
 documentation](https://docs.genians.com/release/ko/monitoring/switch/browsing-switches.html#browsing-switches)
