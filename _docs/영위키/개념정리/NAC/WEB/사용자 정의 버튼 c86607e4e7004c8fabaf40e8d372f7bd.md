    - 버튼을 표시할 대상 선택 (노드그룹에 포함된 경우에만 출력-JYKIM 노드그룹)
        
        ![Untitled](%EC%82%AC%EC%9A%A9%EC%9E%90%20%EC%A0%95%EC%9D%98%20%EB%B2%84%ED%8A%BC/Untitled%201.png)
        
    - 테스트 결과
        
        ![버튼 클릭 시 **링크된 사이트로 연결**](%EC%82%AC%EC%9A%A9%EC%9E%90%20%EC%A0%95%EC%9D%98%20%EB%B2%84%ED%8A%BC/Untitled%202.png)
        
        버튼 클릭 시 **링크된 사이트로 연결**
        
2. **팝업윈도우**

    - 버튼을 표시할 대상 선택 (노드그룹에 포함된 경우에만 출력-JYKIM 노드그룹)

    - 테스트 결과

    - 버튼을 표시할 대상 선택 (노드그룹에 포함된 경우에만 출력-JYKIM 노드그룹)
