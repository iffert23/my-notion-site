### 1. CSV

- ID 생성
- 문자셋 **EUCKR**
- DB 타입 **CSV(Upload)**, 헤더 필드 사용 On
- 데이터소스 구분값 : 다른 CSV 동기화 시에도 삭제되지 않기 위해서 구분 값 임의로 지정
- 사용자, 부서, 직급 정보 파일 업로드
    
    ![Untitled](%EC%A0%95%EB%B3%B4%20%EB%8F%99%EA%B8%B0%ED%99%94%20(editing)/Untitled.png)
    
- 사용자 정보 : ID, 이름, 패스워드(+암호화 알고리즘), 부서ID, 직급ID 컬럼명 입력
    
    ![CSV2.png](%EC%A0%95%EB%B3%B4%20%EB%8F%99%EA%B8%B0%ED%99%94%20(editing)/CSV2.png)
    
- 사용자 용도 설정, 사용자 정보 수정 여부 설정 (수정여부 선택사항)
