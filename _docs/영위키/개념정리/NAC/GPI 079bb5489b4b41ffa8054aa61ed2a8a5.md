# GPI

---

- **참고**

GPI : 내 PC 지키미, 주기적으로 여러 항목에 대해 보안점검을 함. 점검결과에 대해서 사용자별 보안 점수 확인 가능.

CLI 설정 방법은 NAC 센터 설정과 거의 동일함.

최초 접속 : admin / genian

설정 필요 값 : interface ip, default-gateway, name server, management server enable, node server enable, database server (username, enable, password, access list), superadmin (ID, PW만)
모든 설정 끝난 이후에는 reboot

NAC→GPI 이미지 변경(IP가 세팅되어 있는 상태) : shell 접속 → touch /disk/sys/conf/CLOGINPASS (CLOGINPASS 생성) → ssh 접속하여 cd /tmp → rz (파일업로드) → rm -f /disk/sys/conf/CLOGINPASS (CLOGINPASS 삭제) → geniup -cdf (이미지)

초기화 및 shutdown 방법은 NAC와 동일

접속 : https://(서버IP)/mc
