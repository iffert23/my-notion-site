# Genian NAC 5.0

---

- **참고**
    
    **2022년 파트너 NAC5.0 초급교육(1. 제품소개)**
    
- 가시성 확보가 되면 내부 네트워크에 대한 분류/통제가 가능
- PC 내의 다양한 정보제공
    - Agent 설치 시 다양한 정보확인 가능
        
        ![Untitled](Genian%20NAC%205%200/Untitled.png)
        
- 대시보드를 통한 다양한 통계, 상태정보 현황 제공
    - 탭, 위젯 등 여러가지 스타일 있음, 관심 있는 항목만 표현, GUI형식, 기능 활용 쉬움
- 다양한 조건의 분류 가능
    - 100가지 이상의 정보를 그룹화→분류하여 한눈에 통계 확인 가능
    - ⇒그룹핑
- 네트워크 통제
    - 전용장비 이용 ⇒ IP관리통제, 미인증 통제, 방화벽수준 IP/Port/Protocol 기반 통제
    - 전용장비&Agent설치유도 ⇒ 정책 미준수 단말 통제, 무결성 위반 단말 차단
- 단말장치 통제기능 제공
    - PC 내의 정보 수집 및 단말 제어 기능
        
        ![Untitled](Genian%20NAC%205%200/Untitled%201.png)
        
- 유연한 차단페이지
    - 차단사유없이 차단되었다는 메세지만 나오는 것이 아니라 차단 사유를 안내하고 조치방안을 안내하는 페이지를 제공한다.
    - 안내메세지 등 문구 지정 가능, 정책별로 각기 다른 페이지를 지원함, 사용자 조치방법 안내, 다용도 링크 제공
- 제품 주요특징
    - 구성요소 : 이중화 기능 제공
    - 안전성 : Non-Driver 방식. 커널 레벨이 아니라 응용프로그램 레벨에서 돌아가기 때문에 OS 및 타 시스템과의 충돌위험이 거의 없음
    - 에이전트 : 다양한 정보 활용 가능(세밀한 분류표, 여러가지 정보 활용가능)
    - 플랫폼 탐지 : 자동 노드 분류(단말식별/식별정보 온라인 업데이트, 플랫폼 정보 자동 업데이트)
    - 연동 : 전체 단말을 연동해서 보안성을 강화함
    - 단계적 정책, 자동분류 등
    
     ⇒ 효과
    
    - 에이전트 기능 모듈화 : 필요에 따라 기능추가/삭제 가능. 리소스 사용률 낮음
    - 신규 단말 유입 프로세스 : 내부 네트워크에 보안 프로세스 정립 (노드 연결~단말 무결성)
    - 권한 별 네트워크(서비스) 제한 : RBAC(Roll Base Access Control) 기능
    - NAC 기반 InsightsE 추가확장 : 별도 프로세스 기반 엔드포인트 이상유무 탐지, 다양한 침해 분석 가능
- 도입효과
    - 가시성 확보, 다양한 형태의 제어, 단계적 검증 → 내부 네트워크 보안 강화
    - SW통제, 플랫폼 분류, IP관리, 통합 관리, 인증 강화, 네트워크 통제
- 기능활용
