# VXLAN

---

vxlan 검색하면 evpn이 많이 나온다. 

virtual extensible lan → 가상 확장가능 랜

장점 : 확장성 향상, 보안성 향상, 개선된 성능 및 회복성

evpn : 가상 네트워크의 디바이스를 위해 실제 인프라에서 레이어2 연결을 제공하거나 레이어3 라우팅을 활성화 할 수 있다고 함.

=ethernet VPN

overrey network를 위한 MAC 주소 학습 컨트롤 플레인의 역할…. 다양한 데이터 플레인 캡슐화 기술 지원가능. 엄격히 규정되지 않은 네트워크 패브릭에 특히 유용.
