- 서브넷팅(하나의 네트워크를 두 개 이상의 작은 그룹으로 세분화하는 작업)해서 트래픽이 지정된 서브넷 내에서 발생하도록 통제할 수 있음
- 동일한 트래픽이 발생하더라도 정체를 줄일 수 있음
- 하위 네트워크를 사용하면 패킷 이동거리가 줄어들어서 성능이 향상됨.
