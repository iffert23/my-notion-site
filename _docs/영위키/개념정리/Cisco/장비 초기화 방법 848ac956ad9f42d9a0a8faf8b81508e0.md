# 장비 초기화 방법

---

Mode 버튼 누른 상태로 파워 연결 → 5초가량 유지해야 함 (아니면 부팅중에 ctrl + C 연타)

Password Recovery 어쩌구 문구 보이면 정상적으로 부팅 된 거임

flash_init 입력 ← flash 초기화

dir flash: ← flash 디렉토리 확인

vlan.dat, config.text, private-config.text 디렉토리 삭제해야함

delete flash:vlan.dat → Y

delete flash:config.text → Y

delete flash:private-config.text → Y

dir flash: ← 정상적으로 삭제됐는지 확인

reset ← 정상삭제됐으면 부팅 진행

초기 화면으로 로딩됐는지 확인
