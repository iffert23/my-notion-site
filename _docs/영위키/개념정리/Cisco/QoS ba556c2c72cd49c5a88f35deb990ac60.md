QoS(Quality of Service) : 통신 서비스 품질.

네트워크 통신 (Baundwith) 속도를 제어하는 기술

트래픽의 속도나 대역을 임의로 제어함

config 순서

1) access-list 를 만든다

`access-list extend [name] permit [ip] any any`

2) class-map 을 만들고 매칭시킬 access-group을 지정한다

`class-map [name]`

`match [access group/protocol]`

3) policy-map을 만든다

`policy-map [name]`
